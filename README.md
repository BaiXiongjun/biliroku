# biliroku
bilibili 生放送（直播）录制

BiliRoku v1.4.1

Licensed by GPLv3 详情请见LICENSE文件

BiliRoku是用来录制bilibili生放送（直播）内容的。
说录制其实不准确，其实是直接接收直播的视频流并保存到本地。

使用方法：输入房间号。
房间号就是看直播的网址：http://live.bilibili.com/xxxxx
里面xxxxx的数字。

作者：Zyzsdy
（主页 http://zyzsdy.com/biliroku)

运行平台: .NET framework 4.0以上
